# Tech-Bazaar 🛒💻

**Tech-Bazaar** is a developing second-hand marketplace platform focused on buying and selling **tech peripherals and hardware**. The project is built using **Firebase Studio** and aims to provide a simple, secure, and community-friendly environment for users to trade used technology items.

🌐 Live Preview: [https://store.shadow269.in/](https://store.shadow269.in/)

---

## 🚧 Project Status

This project is currently **under active development**.

At the moment, users can:

* Visit the live website
* Sign in using **Google authentication**
* Explore the interface and platform structure

⚠️ Image upload functionality is **currently not working** and is being fixed.

---

## 🎯 Project Goal

Tech-Bazaar is designed to become a dedicated platform where users can:

* Buy and sell second-hand tech peripherals
* List hardware components
* Trade accessories and electronics within the community

The focus is on simplicity, reliability, and accessibility for tech enthusiasts.

---

## 🧰 Built With

This project currently uses:

* Firebase Studio
* Firebase Authentication (Google Sign-In)
* Firebase Hosting
* TypeScript

Future backend enhancements will expand database handling and marketplace features.
